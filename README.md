## Random Drink Generator

Simple webpage that includes:
* An application to generate random ingredients for a drink by choosing the number of the ingredients
* A catalogue that shows different kind of cockails and beverages
* You can search for cocktails by ingredients

Created by using [The Cocktail DB](https://rapidapi.com/theapiguy/api/the-cocktail-db) API by [theapiguy](https://rapidapi.com/user/theapiguy).

This application is only used for excercising purposes.
